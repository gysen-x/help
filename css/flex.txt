flex
: 1 1 50% - контейнер может увеличиваться и уменьшаться в размерах
: 0 0 33.33% - контейнеру запрещено увеличиваться и уменьшаться в размерах

flex-direction - направление потока флекса
: row - слева направо (значение по умолчанию)
: row-reverse - справа налево
: column - сверху вниз
: column-reverse - снизу вверх

flex-wrap - параметр переноса элементов в контейнере
: nowrap - элементы выстроены в одну линию и не переносятся (значение по умолчанию)
: wrap - элементы переносятся сверху вниз если не помещаются по ширине контейнера
: wrap-reverse - элементы переносятся снизу вверх если не помещаются по ширине контейнера

flex-flow - комбинированное свойство, состоящее из flex-direction и flex-wrap
: row nowrap - первое свойство отвечает за направление, а второе за перенос (значение по умолчанию)

justify-content - расположение по отношению к основной оси
: flex-start - элементы прижимаются к началу потока (значение по умолчанию)
: flex-end - элементы прижимаются к концу потока
: center - элементы группируются по центру
: left - элементы прижимаются к левому краю не зависимо от направления потока
: right - элементы прижимаются к правому краю не зависимо от направления потока
: space-between - элементы распределяются по ширине контейнера, при этом крайние прижимаются к краям
: space-around - элементы распределяются по ширине контейнера, у каждого элемента появляется отступ слева и справа
: space-evenly - элементы распределяются по ширине контейнера, отступы space-around не дублируются

align-items - расположение по отношению к второстепенной оси
: stretch - все элементы растягиваются в зависимости от размера самого большого элемента (значение по умолчанию)
: center - все элементы центрируются по второстепенной оси
: flex-start - все элементы прижимаются прижимаются к началу второстепенной оси
: flex-end - все элементы прижимаются прижимаются к концу второстепенной оси
: baseline - все элементы центрирутся в зависимости от содержимого, а не размера элемента

align-content - свойство, которое отвечает за расстояние между строками второстепенной оси
: flex-start(start) -  строки элементов прижимаются к началу потока (значение по умолчанию)
: flex-end(end) - строки элементов прижимаются к концу потока
: center - строки элементов группируются по центру
: stretch - строки элементов растягиваются 
: space-between - строки элементов распределяются по ширине контейнера, при этом крайние прижимаются к краям
: space-around - строки элементов распределяются по ширине контейнера, у каждого элемента появляется отступ слева и справа

gap - выставление гапов вручную
gap: 10px 20px = row-gap: 10px + column-gap: 20px
