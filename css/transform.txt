##### TRANSFORM #####

transform

: translateX(5px); - сместить по горизонтали
: translateY(10px); - сместить по вертикали
: translate(5px, 10px); - сместить в обоих плоскостях

: scaleX(5px); - изменить размер по горизонтали
: scaleY(1.5); - изменить размер по вертикали в полтора раза
: scale(5px, 10px); - изменить размер в обоих плоскостях

: rotate - повернуть

: skewX(-13deg); - наклон по горизонтали
: skewY(-13deg); - наклон по вертикали
: skew(-13deg, -13deg); - наклон в обоих плоскостях
