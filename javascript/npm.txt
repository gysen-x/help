##### N P M #####

npm list - список установленных пакетов

node index.js - исполнить файл index.js с помощью Node

npm install -g - установка пакета глобально для текущего компьютера

npm init - локальная инициализация package.json 

npm init -y - локальная инициализация package.json со всеми утвердительными ответами

node_modules - папка, в которой лежат все зависимости проекта

npx eslint --init - установка пакетов ESLint
