##### O P E R A T O R S   -   О П Е Р А Т О Р Ы #####

##### Арифметические #####

+ сложение
- вычитание
* умножение
/ деление
** степень
% остаток от деления

##### Операторы присваивания #####

++ инкремент (увеличение переменной на единицу)
-- декремент (уменьшение переменной на единицу)

+= присваивание со сложением
-= присваивание с вычитанием
*= присваивание с умножением
/= присваивание с делением
%= присваивание с операцией остаток деления

##### Операторы сравнения #####

== оператор равно
операция выполняется с преобразованием типов, выражение 3 == '3' будет true ('3' будет приведена к 3)

=== оператор строго равно
операция выполняется без преобразования типов, выражение 3 == '3' будет false 

!= оператор не равно (преобразовывает типы)

!== оператор строго не равно (не преобразовывает типы)

> оператор больше чем
< оператор меньше чем 
>= оператор больше или равно
<= оператор меньше или равно

##### Логические #####

&& оператор И
x < 10 && y > 1 это true

|| оператор ИЛИ
x == 5 || y == 5 это false

! оператор НЕ
!(x == y) это true


##### Условный оператор ? #####

let accessAllowed = age > 18 ? true : false;

##### Методы #####

Math.sin (x) - возвращает значение синуса аргумента (!!! радианах от -1 до 1)
Math.cos (x) - возвращает значение косинуса аргумента (!!! радианах от -1 до 1)
Math.tan (x) - возвращает численное значение для тангенса угла (!!! радианах от -1 до 1)

##### Функции #####

Math.min (1, 2, 3) - возвращает минимальное значение из аргументов
Math.max (1, 2, 3) - возвращает максимальное значение из аргументов.  
Math.floor (x) - округление до ближайшего целого в меньшую сторону
Math.ceil (x) - округление до ближайшего целого в большую сторону
Math.abs(x) - возвращает модуль числа
Math.round(x) - округляет число по правилам математики

Math.sqrt(x) - возвращает квадратный корень из аргумента
Math.pow(base, exponent) - возводит число "base" в степень "exponent"
Math.log(x) - вычисляет натуральный (по основанию е) логарифм числа
Math.exp(x) - вычисляет экспоненту - значение числа е в степени аргумента "х"

##### MATH.RANDOM ######

Math.random() - возвращает случайное число с плавающей точкой от 0 (включительно) до 1

Чтобы из данного числа сделать случайное число из указанного диапазона нам понадобится:

Math.floor(Math.random() * max) - где max - это количество значений в диапазоне от 0 до max-1

Math.floor(Math.random() * (max - min) + min) - где min - это минимальное значение, а max - максимальное
