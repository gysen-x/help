##### Объявление переменной #####

var x = 1; 
var studentName = "Vasiliy"; - объявление переменной с помощью var

let x = 1;
let studentName = "Vasiliy"; - объявление переменной с помощью let

Отличие let от var:﻿ Переменная, объявленная через var, видна везде в функции. Переменная, объявленная через let, видна только в рамках блока {...}, в котором объявлена.

##### Значения переменных #####

- Число (number)

- Строка (string)

- Логическая (boolean)

- Неопределенная (undefined)

- Объект (object)

- Пустая (null)
