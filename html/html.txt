##### Технические #####
<!--...--> - комментарий
<!doctype html> - использование html 5 версии
<html>Весь контент</html> - тег, который оборачивает весь html-документ
<head>Хэд</head> - включает метаданные: название документа, ссылки на внешние файлы
<meta charset="UTF-8"> - определяет кодировку символов документа
<title>...</title> - заголовок документа не более 60 символов
<link rel="stylesheet" type="text/css" href="style.css"> - задать стили внешней таблицей стилей
<style> @import url(style.css); </style> - задать стили импортом
<style>.paper {width: 200px;} </style> - задать стили тегом
<p style="color: #666666; background-color: #ef4444; padding: 20px;"> - задать стили атрибутом элемента

##### Структурные #####
<body>Боди</body>
<header>Хедер</header> - идентификация верхней части страницы, может включать заголовок и навигацию
<nav>Навигация</nav> - блок основных навигационных ссылок на странице
<article>Статья</article> - независимый контент, который может применяться в другом месте
<section>Секция</section> - тег для отделения связанного между собой содержимого
<aside>Боковое меню</aside> - боковая панель
<footer>Футер</footer> - тег в конце страницы

##### Строчные #####
<span>Контейнер</span> - не несет никакого симантического смысла, служит для стилизации
<p>Параграф</p> - Абзац
<h1>Заголовок 1</h1> - Заголовок h1 может быть только один на странице 
<h2>Заголовок 2</h2> - Заголовок h2
--"--
<h6>Заголовок 6</h6> - Заголовок h6
<strong>Жирный</strong> - симантический тег, делает текст жирным и делает акцент при голосовом выводе
<em>Курсив</em> - симантический тег, делает текст курсивом и делает акцент его при голосовом выводе
<a href="http://..." target="_blank">Гиперссылка</a> - ссылка на другую страницу в новом окне
<a href="mailto:mail@gmail.com">Email Me</a> - ссылка на почту

##### Блочные #####
<div>Контейнер</div> - не несет никакого симантического смысла, служит для стилизации
